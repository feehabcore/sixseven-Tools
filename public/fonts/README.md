# KUREL FLOW Font Directory

This directory should contain the Bengali font files for the website.

## Required Files

- `LiAdorNoirrit-Regular.ttf` - TrueType font file
- `LiAdorNoirrit-Regular.woff2` - Web font file (recommended for better performance)

## How to Add Fonts

1. Obtain the "Li Ador Noirrit" font files
2. Place them in this directory
3. The font will automatically be loaded via `app/globals.css`

## Fallback

If these font files are not available, the website will automatically fall back to "Noto Sans Bengali" which is loaded from Google Fonts.
